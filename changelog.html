核心原則

  • philosophy
    "好品味"原則：

    消除特殊情況，簡化邏輯

  • build
    實用主義：

    解決實際問題，不過度設計

  • minimize
    簡潔執念：

    函數短小精悍，避免深層嵌套

  • verified
    向後兼容：

    確保現有功能不被破壞

  • speed

  • code

    避免框架依賴，提升載入速度

  • cloud
    Google Apps Script：

    簡化部署流程，降低維護成本

  • integration_instructions
    YouTube 官方 API：

    確保資料準確性和可靠性
