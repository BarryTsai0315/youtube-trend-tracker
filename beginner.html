settings 第一階段：建立資料收集系統 📊

步驟 1：建立 Google Apps Script 專案

1.1 開啟 Google Apps Script

  • 前往：script.google.com
  • 用你的 Google 帳號登入

1.2 建立新專案

  • 點擊「新增專案」
  • 重新命名為「YouTube 趨勢追蹤系統」

1.3 設置程式碼

  • 複製專案中的 src/youtube-search-api.gs 檔案內容
  • 貼上到 Google Apps Script 編輯器中

步驟 2：啟用 YouTube API

2.1 在 Apps Script 中啟用

  1. 點擊左側「服務」（Services）
  2. 點擊「新增服務」➕
  3. 搜尋並新增：YouTube Data API v3

2.2 在 Google Cloud 中啟用

  1. 前往：console.cloud.google.com
  2. 選擇專案 → API 和服務 → 程式庫
  3. 搜尋 YouTube Data API v3 → 啟用

步驟 3：執行初始化

3.1 首次授權

  • 選擇 setupSystem 函數
  • 點擊執行，完成 Google 授權流程

3.2 開始資料收集

  • 執行 dailyYouTubeTracking() 開始收集資料
  • 等待 3-5 分鐘完成首次執行

web 第二階段：使用 Web 分析工具 🌐

工具 1：影片搜尋分析 📺

🆕 v3.1 新功能

✨ 日期範圍篩選器
✨ 觀看數範圍設定
✨ 智能分頁系統
✨ 防抖動搜尋

🎯 主要功能

  • 響應式影片網格：5 列精美卡片展示
  • 多維度篩選：日期範圍、地區、類型、關鍵字、觀看數範圍
  • 即時搜尋：標題和 hashtags 全文搜尋
  • 統計顯示：觀看數、按讚數、留言數、影片時長

📱 使用方法

  1. 在瀏覽器中開啟 影片搜尋分析
  2. 系統自動載入最新資料
  3. 使用篩選器進行數據篩選：
    • 日期範圍：設定資料收集的時間範圍
    • 地區篩選：美國、印度、台灣、巴西、印尼、墨西哥
    • 類型篩選：一般影片、Shorts 短影片
    • 觀看數篩選：設定最小/最大觀看數範圍
    • 關鍵字搜尋：輸入關鍵字搜尋標題和標籤
  4. 使用分頁功能瀏覽大量數據
  5. 點擊影片縮圖直接在 YouTube 觀看

工具 2：Hashtag 統計分析 📊

🎯 主要功能

  • Chart.js 視覺化：圓餅圖、長條圖、環圈圖可切換
  • 雙重排行榜：使用頻率排行 + 總觀看數排行
  • 多維度篩選：時間範圍、地區、類型、排序方式
  • 詳細數據表格：完整統計數據和百分比

📈 使用方法

  1. 點擊頁面頂部「Hashtag 分析」按鈕
  2. 選擇篩選條件：時間範圍、地區選擇、影片類型、排序方式
  3. 切換圖表類型：圓餅圖、長條圖、環圈圖
  4. 查看詳細數據表格了解完整統計

工具 3：標題文字雲分析 ☁️

🎯 主要功能

  • 智能分詞：支援中英文混合文本分析
  • 動態文字雲：字體大小和顏色反映重要程度
  • 停用詞過濾：自動過濾無意義詞彙
  • 互動式設計：點擊關鍵詞查看相關影片

☁️ 使用方法

  1. 點擊頁面頂部「文字雲分析」按鈕
  2. 設定分析參數：時間範圍、地區篩選、最小字數
  3. 查看文字雲結果：字體大小反映頻率，顏色反映觀看數
  4. 參考排行榜：頻率排行和觀看數排行

help 🔧 疑難排解

資料收集問題

❌ 「YouTube is not defined」

→ 確保已啟用 YouTube Data API v3 服務

❌ 「配額已用盡」

→ YouTube API 每日免費額度用完，隔天再試

❌ 「權限不足」

→ 重新執行 setupSystem() 完成授權

Web 工具問題

❌ 網頁顯示「載入資料中...」

→ 檢查 Google Apps Script API 是否部署為網路應用程式

❌ 圖表無法顯示

→ 確保瀏覽器支援 JavaScript

❌ 文字雲空白

→ 確認有足夠的標題資料，調低「最小字數」設定
